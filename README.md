Цей скрипт призначений для автоматизації рутинних процесів обробки результатів біоінформатичного пайплайну MIRA 2.0 для вірусів грипу. Програма готує отримані консенсусні послідовності та метадані для завантаження в базу даних GISAID, проводить аналіз за допомогою зовнішніх інструментів (Nextclade) та автоматично генерує стандартизовані звіти для регіональних Центрів контролю та профілактики хвороб (ЦКПХ).
