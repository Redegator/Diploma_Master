🧬 Обробка результатів MIRA 2.0 (process_mira.py)
Цей скрипт призначений для автоматизації рутинних процесів обробки результатів біоінформатичного пайплайну MIRA 2.0 для вірусів грипу. Програма готує отримані консенсусні послідовності та метадані для завантаження в базу даних GISAID, проводить аналіз за допомогою зовнішніх інструментів (Nextclade) та автоматично генерує стандартизовані звіти для регіональних Центрів контролю та профілактики хвороб (ЦКПХ).
🛠 Залежності та вимоги
Для коректної роботи скрипта у вашому середовищі (conda/venv) мають бути встановлені наступні Python-бібліотеки:

-pandas
-biopython
-openpyxl

Крім того, для виконання третього етапу скрипт потребує наявності зовнішнього оболонкового скрипта run_nextclade.sh у тій самій директорії, а також локально встановленого інструменту Nextclade із завантаженими референсними датасетами.
